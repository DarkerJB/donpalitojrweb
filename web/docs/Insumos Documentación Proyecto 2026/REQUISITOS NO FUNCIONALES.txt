REQUISITOS NO FUNCIONALES

Requisitos No Funcionales (RNF) Específicos.

RNF: Prioridad Alta.

Número del Requisito	RNF-01
Nombre del Requisito	Seguridad
Tipo	Requisito
Fuente del Requisito	Cliente
Prioridad del Requisito	Alta/Esencial
Descripción	El sistema debe implementar medidas de seguridad para proteger contra ataques y accesos no autorizados.

Número del Requisito	RNF-02
Nombre del Requisito	Tiempo de Carga
Tipo	Requisito
Fuente del Requisito	Cliente
Prioridad del Requisito	Alta/Esencial
Descripción	El sistema debe cargar en menos de 3 segundos en una conexión de red estándar.

Número del Requisito	RNF-03
Nombre del Requisito	Disponibilidad
Tipo	Requisito
Fuente del Requisito	Cliente
Prioridad del Requisito	Alta/Esencial
Descripción	El sistema debe estar disponible 24/7 con un tiempo de inactividad mínimo.

Número del Requisito	RNF-04
Nombre del Requisito	Compatibilidad de Navegadores
Tipo	Requisito
Fuente del Requisito	Cliente
Prioridad del Requisito	Alta/Esencial
Descripción	La página web debe ser compatible con los principales navegadores (Chrome, Firefox, Safari, Edge).

Número del Requisito	RNF-05
Nombre del Requisito	Responsividad
Tipo	Requisito
Fuente del Requisito	Cliente
Prioridad del Requisito	Alta/Esencial
Descripción	La interfaz de usuario debe ser completamente responsiva, adaptándose a dispositivos móviles, tablets y escritorio.


Número del Requisito	RNF-06
Nombre del Requisito	Escalabilidad
Tipo	Requisito
Fuente del Requisito	Cliente
Prioridad del Requisito	Alta/Esencial
Descripción	El sistema debe ser escalable para manejar un aumento en el tráfico de usuarios sin afectar el rendimiento.


Número del Requisito	RNF-07
Nombre del Requisito	Cumplimiento Legal
Tipo	Requisito
Fuente del Requisito	Cliente
Prioridad del Requisito	Alta/Esencial
Descripción	La página web debe cumplir con todas las leyes y regulaciones locales sobre privacidad y protección de datos.

Tipo	Requisito
Fuente del Requisito	Cliente
Prioridad del Requisito	Media/Deseado

Nombre del Requisito	Rendimiento
Tipo	Requisito
Fuente del Requisito	Cliente
Prioridad del Requisito	Media/Deseado

Nombre del Requisito	Recuperación
Tipo	Requisito
Fuente del Requisito	Cliente
Prioridad del Requisito	Media/Deseado

Tipo	Requisito
Fuente del Requisito	Cliente

Fuente del Requisito	Cliente
Prioridad del Requisito	Media/Deseado

Número del Requisito	RNF-14
Nombre del Requisito	Actualizaciones y Parcheo
Tipo	Requisito
Fuente del Requisito	Cliente
Prioridad del Requisito	Media/Deseado

Tipo	Requisito
Fuente del Requisito	Cliente

Tipo	Requisito
Fuente del Requisito	Cliente

Tipo	Requisito
Fuente del Requisito	Cliente
